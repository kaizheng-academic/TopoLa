# Seurat+TopoLa 


Installation
------------

You need to install Seurat first. Click [here](https://satijalab.org/seurat/)  for detailed installation instructions.


## Datasets 

You need to click [here](https://drive.google.com/file/d/1k3AD1tdELfYM1xXs9jjhKJ6nb1pu71As/view?usp=share_link)  to download the relevant dataset.


How to use `demo.R`
-------------------
To directly use demo.R, you can run the following code:
```R
Rscript demo.R
```
